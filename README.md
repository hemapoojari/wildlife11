 🐾 Wildlife Monitoring System
 
An intelligent **multi-modal wildlife monitoring system** built using **Python and Streamlit**.
This system helps analyze wildlife data, predict migration risks, identify animals from images, and generate conservation alerts to support wildlife protection.
---
🚀 **Features**

 📊Overview Dashboard

Displays key wildlife statistics such as:

* Total species count
* Population levels
* Threat levels

📈 Analytics Dashboard

Visual insights including:

* Population distribution
* Threat level distribution
* Wildlife trend analysis

 🔮 Migration Risk Prediction

Machine Learning model that predicts **migration risk levels** using:

* Population size
* Threat level

 🖼 Animal Image Identifier

Upload wildlife images to automatically identify species using:

* Custom trained CNN model
* Pretrained **ResNet50 ImageNet model**

📝 Ranger Report Analysis

Performs **sentiment analysis** on ranger reports to determine wildlife safety levels.

🚨 Conservation Alerts

Automatically generates alerts for:

* High threat level species
* Critical conservation situations

🛠 Installation

1️⃣ Clone the repository

```bash
git clone https://github.com/hemapoojari/wildlife11.git
cd wildlife11
```

 2️⃣ Install required libraries

```bash
pip install -r requirement.txt
```

3️⃣ Ensure required files exist

Required files:

```
data/wildlife_data.csv
model/models/
model/image_model.h5
model/image_classes.json
```

---

▶️ Running the Application

Start the Streamlit app:

```bash
streamlit run app.py
```

Open the browser link generated by Streamlit.
